HTML

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Define la estructura básica de una página web mediante etiquetas que organizan el contenido en encabezados, párrafos, listas, enlaces, imágenes, tablas, formularios, entre otros elementos.

css

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación y el diseño de un documento HTML o XML. Mientras que HTML se encarga de la estructura del contenido (como texto, imágenes, videos, etc.), CSS se ocupa de cómo ese contenido se verá visualmente en una página web. Esto incluye aspectos como colores, fuentes, márgenes, alineación, espaciado, y la disposición de los elementos en la página.

Etiqueta "a"

La etiqueta en HTML es utilizada para crear hipervínculos, que permiten a los usuarios navegar de una página web a otra o a diferentes secciones de la misma página. "a" proviene de "anchor" (ancla en inglés), porque originalmente se usaba para definir "anclas" o puntos de referencia dentro de un documento.

a href=""

Etiqueta img

La etiqueta img en HTML se utiliza para insertar imágenes en una página web. A diferencia de otras etiquetas, img es una etiqueta vacía, lo que significa que no tiene una etiqueta de cierre como

img src="" alt=""

Texto HTML

El texto en HTML es cualquier contenido escrito que aparece entre las etiquetas HTML en un documento web. HTML (HyperText Markup Language) se utiliza principalmente para estructurar y organizar este contenido textual en una página web.

P h1 h2 h3 h4 h5 h6

Proceso de diseño

El proceso de diseño en HTML es una serie de pasos que se siguen para planificar, estructurar y desarrollar una página web utilizando HTML (y generalmente CSS y JavaScript). Este proceso no solo involucra la codificación, sino también la planificación y la consideración de la experiencia del usuario, la estructura de la información y la apariencia visual.

¿Qué funcionalidades son importantes (formularios, enlaces, multimedia)?
